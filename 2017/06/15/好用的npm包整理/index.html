Surge

Surge 是一个静态资源部署工具，支持一行命令部署静态资源到 project.surge.sh，只需要指定需部署的文件夹以及二级域名

now

now 是 zeit 出品的命令行一键部署工具，支持部署 node 项目以及静态网站项目
